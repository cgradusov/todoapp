# TodoApp  

TodoApp is a React application created for frontend skill demonstration.   

## Instalation  
Use the package manager to install dependencies.  

```bash
npm i
```  
or  
```bash
yarn
```  

## Starting project  

```bash
npm start
```  
or  
```bash
yarn run start
```
